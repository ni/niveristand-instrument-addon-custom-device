Instrument Protocol Addon

This device allows the user to define instruments commands generically. The communication structure should be command response. Meaning that the instrument doesn't initiate any of the communications. It is designed for 10s of Hz communication rates.

A quick start document provides an overview of the addon capabilities. It is located at:

"%Public%\Documents\National Instruments\NI VeriStand (Year)\Custom Devices\Instrument Addon\Windows\Quick Start Documentation\Instrument Addon Quick Start.md"

Most of the addon sections contain additional built-in documentation.

What do you want to do?

Disable the Device

Right click on the custom device and select Disable.

Add Messages

To add a new message, you must first have a message group. You add this by right clicking on the custom device and selecting Add New Message Group. A message group contains any number of messages. An example might be having a 2 channel power supply. You could then have 2 groups, one for each channel. Each group would contain the same messages, just configured for each channel. After adding the message group, add a message by right clicking on the message group and selecting Add Message.

Export the Current Configuration

The custom device configuration can be exported to a file. This allows the user to re-add the custom device later without having to manually build up the command set. To get this configuration file, right click on the custom device and select Export Config.
