Command

This section contains the command equation. Type in the equation and use [] to mark variables. As an example, a command VRANGE[CH]:[VALUE] would have variables CH & VALUE. This would allow the user to set a voltage range on a specific channel. If there isn't a command required, delete this section.

What do you want to do?

Create a Command

You create a command by entering the text (case sensitive) into the Command field. This text defines what strings and VeriStand data get sent to the instrument. If desired, you can make the command static. As an example, you can enter VSET1:10.000. This would set the voltage range on channel 1 to 10 volts. However, if you do this, you would need to create the same message for any additional channels or ranges (ie. VSET2:10.000). Instead, you can parameterize the commands with variables. Use [] to mark the variables in the command. Then link those variables to your metadata channels in your custom device through the Linkages table. Do this by highlighting the variable you would link to link and then selecting the desired metadata channel from Available MetaData Channels.

Create a Command Variable

Command variables are refered to as metadata. To create a meta data channel, right click on the desired Command or on Global MetaData Channels and select Add MetaData Channel.

No Command Required

If no command is required, delete the command section. Right click on the section and select Delete.

Force a Message Length

Some protocols require messages to always be a certain length. If your instrument requires this, enable Pad. Then set the desired Length and Pad Value.

Note: Length is total message size including any message protection. So, if the message is twenty four bytes of data and two bytes of protection, enter twenty six as the length.
