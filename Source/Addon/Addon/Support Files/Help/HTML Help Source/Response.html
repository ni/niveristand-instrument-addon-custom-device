Response

This section contains the response equation. Type in the equation and use [] to mark variables. As an example, a command VRANGE[CH]? with a response [RANGE] would have response variable RANGE. If there isn't a response required, delete this section.

What do you want to do?

Create a Response

You create a response in the same way you create a command. Enter the text (case sensitive) into the Response Format field. The primary difference is this format defines how to parse data out of the response string and into VeriStand data channels where commands are the reverse. Use [] to mark the variables in the response. Then link those variables to your metadata channels in your custom device through the Linkages table. Do this by highlighting the variable you would link to link and then selecting the desired metadata channel from Available MetaData Channels.

Create a Response Variable

Response variables are refered to as data. To create a data channel, right click on the desired response section and select Add Data Channel.

Define Wait Type

You can define how long to wait for a response either in seconds or in number of bytes. A wait type of Wait sec will wait that amount of seconds, read the available bytes at the port, and read all the available data. Byte Count will wait up to 10 seconds for the selected amount of data to be available, then read just that amount of data. Set Wait Type and relating Value accordingly.

Note: You can use a channel for the wait value. This allows you to change the number of bytes or wait time on the fly. To do this, enable Use Channel? and select the desired channel.

No Response Required

If no response is required, delete the response section. Right click on the section and select Delete.
