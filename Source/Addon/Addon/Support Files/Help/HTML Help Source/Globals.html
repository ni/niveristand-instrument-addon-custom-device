Globals

This section contains all the global inputs that can be used in commands. Define any command parameters here that can be re-used in multiple messages. As an example, most instruments are multiple channel. The commands for these devices require the channel number as part of the command. Therefore, you could create a global metadata channel named CHAN with a format of %d. This would then be used in any messages that require a channel. IE. VSET[CHAN]:10.000 would set the voltage output to 10 on whatever CHAN is selected.
