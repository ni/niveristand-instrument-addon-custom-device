MetaData Channel

This section allows the user to define the way to format the particular variable into a string.

What do you want to do?

Set MetaData Channel Format

This channel value will be converted to the selected type and added to any configured string. The Channel Type options are Enumerated List, Format and Type Cast.

The enumerated list converts the numeric value of the metadata channel to an associated string value. It is 0 indexed. As an example, if you have the command OUT:[ENABLE], where ENABLE needs to be "OFF" or "ON". You can set the list to OFF, ON. When the message is triggered, if the channel's numeric value is 1, OUT:ON will be sent. If it is 0, OUT:OFF will be sent.

The channel type of format will convert the metadata numeric value to a numeric string of that format. As an example, a Format String of %x will convert the numeric value to a hexidecimal string of that value (ie. a value of 10 will be "A" in the command string).

The channel type of type cast will convert the numeric value of the metatdata channel to its binary representation. So, if you wanted to type cast a double, it would be converted to 8 bytes of binary data. Valid options are DBL, SGL, U64, I64, U32, I32, U16, I16, U8, I8.
