Message Group

This section is simply an organizational element for commands. It allows the user to group commands logically. As an example, you could group messages by channel (ie CH1, CH2) or by type (ie. Config, Acquire, etc).

Create a Message

To create a new message, right click on the message group and select Add Message. Then configure the Command and Response sections as needed. If you are not expecting a command or a response, delete those sections.
